# Summarize

Summarize the selected or visible content. Prefer selected text when available; otherwise use the visible page content. Focus on the key points, important details, and practical takeaways. Keep the summary concise and easy to scan.

Do not invent details that are not supported by the selected text or visible page. If the context is incomplete, summarize only what is available.
